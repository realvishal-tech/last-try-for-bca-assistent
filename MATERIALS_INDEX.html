🐛 Troubleshooting

❌ Materials not showing?

  • Check Firebase credentials in firebase-config.js
  • Verify database rules allow .read: true
  • Check browser console for Firebase errors

❌ Admin login fails?

  • Verify exact email: admin@bca.com
  • Verify exact password: Admin@@2023
  • Check sessionStorage in browser dev tools

❌ Real-time not working?

  • Check Firebase listener is active in console
  • Verify database has materials reference
  • Reload page to sync
