## 🔄 Customization Tips

### Add Your Own Content
1. Login to admin panel
2. Click "Add New Content"
3. Fill in details:
   - Title (e.g., "Linear Algebra Basics")
   - Subject (select from dropdown)
   - Type (Notes, PYQs, Videos, Programs, Lab)
   - Description
   - Link or inline content
4. Click Save

### Change Brand Name
In `index.html`, find and replace:
- "BCA Study Hub" → Your name
- "LND College" → Your college name

### Change Color Scheme
In `style.css`, modify CSS variables:
- `--primary: #4F46E5;` → Your color
- `--secondary: #9333EA;` → Your color
- `--accent: #22C55E;` → Your color

### Add More Subjects
In `script.js`, modify the `SEMESTERS` object to add more subjects.
