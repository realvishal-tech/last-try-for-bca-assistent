QUICK START GUIDE
═════════════════════════════════════════════════════════════════════════════

STEP 1: CREATE FIREBASE PROJECT (5 minutes)
1. Go to firebase.google.com
2. Click "Get Started"
3. Create new project
4. Enable Realtime Database
5. Copy Firebase configuration

STEP 2: UPDATE CONFIGURATION (2 minutes)
1. Open firebase-config.js
2. Replace firebaseConfig with your credentials
3. Save file

STEP 3: TEST LOCALLY (5 minutes)
1. Open admin-materials.html
2. Login with: admin@bca.com / Admin@@2023
3. Add a test material
4. Open index-materials.html
5. See material appear instantly!

STEP 4: DEPLOY (5 minutes)
1. Push to GitHub
2. Connect to Netlify
3. Deploy!
